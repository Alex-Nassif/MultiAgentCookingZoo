This file extends upon the EpyMARL framework and edits it to support multi agent reinforrcement learning with the CookingZoo PettingZoo environment. The PettingZoo environment needs to be installed spearately as linked here: https://github.com/DavidRother/cooking_zoo

The main wrapper is implemented with the test.py file with changes being made to the EpyMARL framework to make the multi agent environment run properly. The license for using EpyMARL is also included. Wrapper is implemented inside the envs folder in the epymarl folder.
